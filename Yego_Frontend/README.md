# GDSC Website

## Environment Setup
```bash
npm install
```

## File Structure
```
src
├─ index.tsx
├─ index.css
├─ App.tsx
├─ logo.svg
├─ api
│  ├─ functionName.ts
|  └─ ...
├─ components
│  ├─ ComponentName
|  |  ├─ index.tsx (.jsx)
|  |  └─ index.scss (.css)
|  └─ ...
├─ states (Optional)
│  ├─ ComponentNameState
|  |  └─ index.tsx (.jsx)
|  └─ ...
├─ ui
│  ├─ ElementName
|  |  ├─ index.tsx (.jsx)
|  |  └─ index.scss (.css)
|  └─ ...
└─ utils
   ├─ name.ts
   └─ ...
```
TypeScript is recommended, but JavaScript is OK.
